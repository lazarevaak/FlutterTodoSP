# Лабораторная работа: "Advanced Todo App" (60 минут)

**Цель:** Реализовать персистентное хранение данных в приложении, экран добавления задач и логику удаления.

## Описание задания
Вам дан файл-шаблон `lib/task_template.dart`.
Сейчас приложение запускается, но список задач пуст, а кнопки не работают (или показывают "заглушки").

## Условия сдачи
1. **Базовая оценка:**
   - Реализовать интерфейс `TodoRepository` используя Hive или Drift.
   - Реализовать экран добавления задачи (`AddTodoPage`).
   - Реализовать удаление задачи с диалогом подтверждения (`AlertDialog`).
2. **Дополнительный балл:**
   - Реализовать ОБЕ реализации БД (Hive и Drift) с возможностью переключения.

---

## Инструкция

### Шаг 1: Подготовка
1. Откройте файл `lib/task_template.dart`.
2. Запустите приложение.
3. Убедитесь, что оно работает (показывает пустой список).

### Шаг 2: Реализация UI
1. **Добавление:** Создайте виджет `AddTodoPage`. Добавьте текстовое поле и DatePicker. Свяжите кнопку FAB на главном экране с переходом на этот экран.
2. **Удаление:** В `HomePage`, при нажатии на иконку корзины, покажите `showDialog` с вопросом "Удалить задачу?". Если пользователь нажмет "Да", вызовите метод репозитория.

### Шаг 3: Реализация БД (Hive или Drift)
1. Выберите базу данных (зависимости уже в `pubspec.yaml`).
2. Напишите класс-наследник `TodoRepository`.
3. Реализуйте все методы интерфейса (`init`, `getTodos`, `addTodo`, `deleteTodo`, `toggleTodo`).

### Шаг 4: Сборка
1. В функции `main()` замените `EmptyRepository` на вашу реализацию.
   ```dart
   final repository = HiveTodoRepository();
   await repository.init();
   runApp(TaskApp(repository: repository));
   ```
2. Перезапустите и проверьте работу всего функционала.
